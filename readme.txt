Kiyoh review score widget

== Description ==
Kiyoh reviews widget. Vertoond de score van de website of webshop in Kiyoh in een widget en gebruik hiervoor de Schema.org opmaak.

== Installation ==
1. Upload or extract the kiyoh-review-widget folder to your site\\\'s /wp-content/plugins/ directory. You can also use the Add new option found in the Plugins menu in WordPress.
2. Enable the plugin from the Plugins menu in WordPress.
3. You will find the Kiyoh review widget under -> appereance -> widgets
4. Fill in the connectorcode and company id and your good to go.
